Setup Instructions

1

Create Telegram Bot

Start a conversation with @BotFather on Telegram and create a new bot to get your API token.

/newbot → Choose bot name → Get API token
2

Configure Channels

Create your movie channel and group where the bot will operate.

# Required IDs:
MOVIE_CHANNEL_ID = -100123456789
MOVIE_GROUP_ID = -100987654321
3

URL Shortener API

Register for a URL shortener service and add your API key.

# bit.ly API example
URL_SHORTENER_API = "your_api_key"
4

Host and Deploy

Deploy your bot on a hosting service or run it locally.

# Python deployment example
python3 bot.py
